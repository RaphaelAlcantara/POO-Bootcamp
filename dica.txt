Qual collection usar: SET, List ou MAP ?
Set - Está diretamente relacionada com a idéia de conjuntos. Assim como um conjunto, as classes que implementam
esta interface não podem conter elementos repetidos.

List - Também chamada de seqüência. É uma coleção ordenada, que ao contrário da inferface Set, pode
conter valores duplicados.

Map - Relação de chave-valor entre os elementos. Cada chave pode conter apenas um único valor associado.
